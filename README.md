# Oxylabs (oxylabs)
Oxylabs is a web intelligence platform providing premium proxy networks, web scraping APIs, and data collection tools for large-scale web data extraction. It includes the Web Scraper API (Realtime and Push-Pull), Dashboard API for usage and statistics, and the Residential Public API for sub-user and statistics management.
